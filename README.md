Student at Pinewood School (Los Altos Hills, CA), graduating **May 2026**. Software head/lead at [FRC Team 4765 PWRUP](https://github.com/PinewoodRobotics).

## Current projects

- **B.L.I.T.Z** — Raspberry Pi swarm for robotics backend code (formally the full system moved into local folders); tooling to build and deploy decentralized, load based programs across machines. — [BlitzImageModifier](https://github.com/godbrigero/B.L.I.T.Z.)

- **Autobahn** — Decentralized pub/sub messaging; client-side RPC; libraries for Rust, Java, Python, TypeScript. — [autobahn](https://github.com/godbrigero/autobahn) (main Rust backend server) - [autobahn_client](https://github.com/godbrigero/autobahn_client) (client implementations)

- **2026 Rebuilt** — Competition robot code with vision (apriltag detection/processing), global positioning (custom EKF impl, GTSAM future), decentralization (auto pi load handling on deploy); (tied to the B.L.I.T.Z stack) — [2026Rebuilt](https://github.com/godbrigero/2026Rebuilt)

- **Viz3** — Visualize pub/sub robotics traffic from the Pi network in a game-style 3D view. — [viz3](https://github.com/godbrigero/viz3)

- **GooglePotato** — Java bridge to Google Cartographer for SLAM with 2D/3D lidars. — [GooglePotato](https://github.com/godbrigero/GooglePotato)

- **command-robot-base** — Open FRC robot template bundling the B.L.I.T.Z stack with regular WPILib command base template. — [command-robot-base](https://github.com/godbrigero/command-robot-base)

- **Unilidar SDK** — Rust and Java rewrites of vendor lidar SDKs so teams aren’t stuck on the original C++ bindings. — [Unilidar-SDK-Rust](https://github.com/godbrigero/Unilidar-SDK-Rust) - [unitree_lidar_sdk_for_java](https://github.com/godbrigero/unitree_lidar_sdk_for_java)

- **CudaTags** — GPU AprilTag detection for Jetson-class hardware; much higher throughput than CPU-only pipelines. — [CudaTags](https://github.com/godbrigero/CudaTags)

- **Napoleon** — Rust path planning (`RRT*`, `A*`) with Java bindings for autonomous follow-through on robots. — [napoleon](https://github.com/godbrigero/napoleon)

- **Pathfinder3D** — Drop-in Java library for 2D/3D grid pathfinding without hand-rolling A\*. — [Pathfinder3D](https://github.com/godbrigero/Pathfinder3D)

- **no-incode-comments** — Rust proc macro: keep documentation in external Markdown instead of inline doc comments. — [no-incode-comments](https://github.com/godbrigero/no-incode-comments)

- **PWRUPWeb / PWRUPCore** — Team site (pinewood.one) and shared Java/Protobuf utilities for robot code. — [pwrup-website](https://github.com/godbrigero/pwrup-website) - [PWRUPCore](https://github.com/godbrigero/PWRUPCore)

- **Campify** — School counseling app: summer programs, login, AI-assisted content filtering, and camp chat (Next.js). — [Campify](https://github.com/godbrigero/Campify)

Also: contributor a while back on **[groq-api-rs](https://github.com/godbrigero/groq-api-rs)** (Rust client for Groq’s API)
